DeckX FORMAT

DeckX turns HTML presentations into exchangeable files

DeckX is a new package format. It bundles HTML slides, speaker notes, styles, image assets, and metadata into one zip-based .deckx file that the player opens with the same runtime contract. 
my-talk.deckx
  manifest.json
  slides/
    001-intro.html
    002-demo.html
  notes/
    001-intro.md
  styles/
    theme.css
  assets/
    hero.png
File boundaryBundle manifest.json, slides, notes, styles, and assets into a single .deckx zip package.
URL inputThe player reads ?deck=<url> and loads any DeckX deck into the same runtime.
Safe defaultA normal DeckX package is a script-free Safe Deck, and speaker notes stay out of the slideshow view.
VerifiableSchema, asset paths, notes coverage, and visual overflow can be checked before a deck is presented.

DeckX BASICS

Four questions that define DeckX

DeckX is a new format, so the words need to be clear first. Presenters do not need a new editor; they need to understand how HTML presentations can be opened and operated like files.
What is DeckX?DeckX is the name of the format and runtime for packaging HTML-based presentations as exchangeable presentation files.
What is DeckX Player?DeckX Player is a browser-based runtime that opens .deckx HTML presentation packages with preview, presenter view, slideshow view, speaker notes, and motion controls.
What is a DeckX package?A DeckX package is a zip-based .deckx file containing manifest.json, HTML slides, Markdown speaker notes, styles, and assets.
Why is it useful?You can present HTML motion and layout directly, without rebuilding it in PPT, while sharing notes and assets as one file or URL.
Keep HTML expressionPreserve CSS motion, data flow, and interactive layout instead of flattening them into images.
Presentation operations includedPreview, presenter view, slideshow view, notes, timer, and motion controls are handled by the player.
Shareable fileSend a .deckx zip or URL and reopen it in the same runtime.
